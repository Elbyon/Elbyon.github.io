Framework CSS

Cela fait maintenant 2 ans que je réalise une veille sur les différents framework CSS. Dans cette rubrique, je vais vous présenter comme s'articule cette veille et à quoi peut-elle me servir.

Définition

Un framework CSS est une bibliothèque permettant une conception Web plus simple et plus conforme aux normes en utilisant le langage des feuilles de style en cascade. La plupart de ces frameworks contiennent une grille et/ou plusieurs éléments de style.

Source : Wikipedia
