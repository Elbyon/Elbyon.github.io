Gedima'Gination

01 - Description du projet

Qu'est-ce que Gedima'Gination ?

Gedima’Gination est un jeu concours organiser par Negomat, nouvel adhérent du réseau Gedimat. Durant ce projet, nous devions développer la totalité de la solution composée d’une application web et d’une application mobile sans support existant.

02 - Environnement de développement

Notre environnement

Le cahier des charges nous imposait certaines technologies. Nous devions réaliser une application mobile à l’aide d’Android Studio avec une base de données intégrée SQLite. Une application web communiquant à une base de données MySQL. Pour finir, la gestion de projet devaient s’effectuer avec GitHub
